# Echofinder 2

## Overview

Echofinder 2 is the second iteration of the Echofinder project. It is a Telegram bot written in Python that uses RAG to find messages in a conversation that relate to a given query.

## Tech Stack

- Python
- Telegram
- PyTelegramBotAPI
- LangChain
- ChromaDB
- GitHub Actions
- Docker
- Heroku
